# Project _Printf

Welcome to the README of my _printf function

## Description

This allows you to display text on the screen window that follows a format defined by the programmer. Its general syntax is:

printf("format", arg_1, arg_2, ..., arg_n);

# Requirements

   


# Supported Specifiers
%c : print a character
%s : print a string
%% : print a percent sign
%d : print a entire
%f : print a decimal number

\ \ : to display a single character 
 \n : to create a line break

# Version Limitation

No full support for all flags (+, #, , etc.)

No width, precision, or length (l, h, etc.) support

No advanced error handling

### Dependencies

Ubuntu 20.04 LTS
C-89
gcc, using the options -Wall -Werror -Wextra -pedantic -std=gnu89
